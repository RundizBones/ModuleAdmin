validateRoleActions()

ActionsController.php : 325

Validate role(s) and action.

protected validateRoleActions( $userrole_ids : string , $action : string ) : array

It's validating roles and action must be selected.
Check that selected role's priority is not in restricted roles.
Check that deleting user default role on register. (no error message just return 'deleteUserDefaultRole' array key in boolean).
This method set http response code if contain errors.
This method was called from indexAction(), doDeleteAction() methods.

Parameters
$userrole_ids : string

The selected role ID(s).

$action : string

The selected action.

Return values
array —

Return associative array with keys:
action The selected action.
actionText The text of selected action, for displaying.
userrole_ids The selected role IDs.
userrole_id_array The selected role IDs as array.
formResultStatus (optional) If contain any error, it also send out http response code.
formResultMessage (optional) If contain any error, it also send out http response code.
formValidated The boolean value of form validation. It will be true if form validation passed, and will be false if it is not.
deleteUserDefaultRole (optional) If deleting user default roles then it will be true.
deleteUserDefaultRoleIds (optional) If deleting user default roles, this will contain the ids.

isUserLoggedIn()

SessionsTrait.php : 42

Check if user is logged in.

protected isUserLoggedIn( [ $user_id : int = null ] [, $userlogin_session_key : string = '' ] ) : bool

After called this method and it was true then you can access total sessions via totalLoggedInSessions property.
After called this method and cookie is valid then you can access cookie data via userSessionCookieData property.
If there is simultaneous login and user's setting is something that is not allowed (such as logout previous, logout all), it will be process here.

Parameters
$user_id : int = null

The user ID. Set to null (default) to auto detect from cookie.

$userlogin_session_key : string = ''

The logged in session key. Set to empty string (default) to auto detect from cookie.

Return values
bool —

Return true if logged in, false for not.
