get()

get(array|string  $name, array|string  $default = '') : array|string

Get config value(s).

Example for get single config name:

$ConfigDb->get('config_name', 'default_value');

Example for get multiple config name:

$names = ['config_name1', 'config_name2'];
$defaults = ['default_value1', 'default_value2'];
$ConfigDb->get($names, $defaults);

Parameters

array|string $name

The config name can be string (name) or array (names).
array|string $default

The default config value if this configuration is not exists or have no value. This can be any types but if name is array then this will always be array.

Throws

\InvalidArgumentException

Throw the error if $name is not string and $name,$default` is different type. If one is array the other one must be array.

Returns

array|string —

Return config value.
If name is array then it will always return the array with name in the key. If config was not found then it will return default value matched in the array index.
If name is string then it will data type as value contain in db. If config was not found then it will return $default argument.

\Rdb\Modules\RdbAdmin\Models\ConfigDb
