generateKeyWithWaitTime()

generateKeyWithWaitTime(integer  $user_id, string  $keyFieldName, string  $timeFieldName, integer  $waitMinute = 10, array  $options = array()) : array

Generate key with wait time.

If the key was generated and still within wait time then it will be use the old one, otherwise it will be generate new.
This method did not create/update/delete the data in DB.

Parameters

integer $user_id

User ID.
string $keyFieldName

Name of field that contain key in field_value.
string $timeFieldName

Name of field that contain time in field_value.
integer $waitMinute

Minutes to wait.
array $options

The associative array options:
'keyLength' (int) The key length to generate. Default is 8.
'keyCharacters (string) The key string to generate. Default is empty string means 0-9, a-z.

Returns

array —

Return associative array with 'readableKey', 'encryptedKey', 'keyTime', 'regenerate' keys.
